# Intelligent Business Dashboard
A dummy business intelligence dashboard that has a filter functionality and displays data. 

## Features

Data table
Dropdown filter
Filter ←→ Table interaction
Filter ←→ Filter interaction

## Stack
 Next js with Typescript
